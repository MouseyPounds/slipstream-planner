About

This app provides a simple way to plan out what skill points to spend when levelling a Slipstream character. Select your class using the menu and distribute points with the buttons. You can optionally name your character build and create a link to share that build with ohers.

The app is written in Javascript and uses jQuery; it is hosted on GitHub Pages at https://mouseypounds.github.io/slipstream-planner/ and the source code repository is https://github.com/MouseyPounds/slipstream-planner. Please report any bugs, suggestions, or other feedback to @mouseypounds on Discord.
